## [Solution](https://leetcode.cn/problems/evaluate-reverse-polish-notation/description/)

```java
class Solution {
    public int evalRPN(String[] tokens) {
        Deque<Integer> stk = new ArrayDeque<>();

        for (String s : tokens) {
            if (s.equals("+")) {
                int a = stk.pop();
                int b = stk.pop();
                stk.push(a+b);
            } else if (s.equals("-")) {
                int a = stk.pop();
                int b = stk.pop();
                stk.push(b-a);
            } else if (s.equals("/")) {
                int a = stk.pop();
                int b = stk.pop();
                stk.push(b/a);
            } else if (s.equals("*")) {
                int a = stk.pop();
                int b = stk.pop();
                stk.push(a*b);
            } else {
                stk.push(Integer.parseInt(s));
            }
        }

        return stk.pop();
    }
}
```
